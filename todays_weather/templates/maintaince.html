Sensors

Sensors located at key points on each turbine send various types of data back to the wind farm’s maintenance team. This data includes information on lubrication levels, vibration, temperatures, and foundation displacement, all of which can be used to plan maintenance. For instance, if the wind eddy sensors show too much vibration in the turbine shaft, it can indicate the shaft has run too far out of place and needs to be realigned.
